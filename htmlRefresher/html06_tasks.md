# **Lesson 6 Tasks: Tables**

## **Setup Instructions**

1. Create folder `html-lesson6`
2. Initialize Git repository
3. Create files as specified
4. Test tables in browser
5. Push to GitHub

---

## **Task 1: Class Schedule**

**Create:** `schedule.html`

Make a weekly class schedule table:
- Caption: "My Class Schedule"
- 6 rows (time periods)
- 6 columns (days + time column)
- Use `<th>` for day headers
- Use `<th scope="row">` for time periods

---

## **Task 2: Price Comparison**

**Create:** `prices.html`

Compare 3 products with table showing:
- Product names
- Prices
- Features (3-4 per product)
- Use thead, tbody
- Add caption
- Use th for headers

---

## **Task 3: Sports Statistics**

**Create:** `stats.html`

Create a sports statistics table:
- Player names
- Games played, points, assists
- At least 5 players
- Use tfoot for totals row
- Add caption explaining data

---

## **Task 4: Contact Directory**

**Create:** `contacts.html`

Employee directory table with:
- Name, Department, Email, Phone
- At least 8 employees
- Group by department using rowspan
- Proper table sections

---

## **Task 5: Complex Table**

**Create:** `complex.html`

Create a table using:
- Both colspan and rowspan
- At least 4x4 grid
- Caption
- All three sections (thead/tbody/tfoot)
- Scope attributes

Example: Monthly budget or meal plan

---

## **Task 6: Data Dashboard**

**Create:** Multi-page site with tables:
1. `index.html` - Overview
2. `sales-data.html` - Sales table
3. `inventory.html` - Inventory table

All pages share navigation.
Each data page has a detailed table with caption, sections, proper headers.

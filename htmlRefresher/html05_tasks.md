# **Lesson 5 Tasks: Lists**

## **Setup Instructions**

1. Create folder `html-lesson5`
2. Initialize Git repository
3. Create files as specified
4. Test in browser
5. Push to GitHub

---

## **Task 1: Shopping List**

**Create:** `shopping-list.html`

Make a grocery shopping list page with:
- H1: "Grocery Shopping List"
- 3 unordered lists for categories:
  - Produce (5 items)
  - Dairy (4 items)
  - Pantry (6 items)
- Use H2 for each category

---

## **Task 2: Recipe with Steps**

**Create:** `recipe-steps.html`

Create a recipe page with:
- Unordered list for ingredients
- Ordered list for cooking instructions
- At least 8 ingredients
- At least 6 numbered steps
- Use figure with image

---

## **Task 3: Navigation Practice**

**Create:** 3 pages with navigation:
1. `index.html`
2. `about.html`
3. `services.html`

Each page must have same navigation using lists:
```html
<nav>
  <ul>
    <li><a href="index.html">Home</a></li>
    <li><a href="about.html">About</a></li>
    <li><a href="services.html">Services</a></li>
  </ul>
</nav>
```

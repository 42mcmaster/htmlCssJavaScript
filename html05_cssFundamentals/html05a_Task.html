Three CSS Methods Practice

This task practices all three CSS methods: inline, internal, and external.

CSS selectors let us target specific elements. We can use element selectors, class selectors, or ID selectors.

Element selectors target all tags of one type, like all <p> tags.

Class selectors use a dot (.) and can be applied to many elements.

ID selectors use a hash (#) and should target just one unique element.

Master these three methods and selectors before moving to the next lesson!

# HTML14: Capstone Project Files

This directory contains all the lesson materials for the HTML14 capstone project: **Multi-Page Responsive Website**.

## Files Included

### Documentation & Planning (for instructors and students)

1. **html14_CapstoneSpec.md** (321 lines)
   - Complete project specification and requirements
   - Project overview, goals, and detailed checklist
   - 10 project theme suggestions
   - 6-session timeline with milestones for each class
   - Submission requirements and tips for success
   - Resources and academic integrity statement

2. **html14_PlanningTemplate.md** (360 lines)
   - Fillable planning document for students
   - Site overview, sitemap, and wireframe planning
   - Color palette and typography selection
   - Content planning (images, text, tables, forms)
   - Navigation and responsive design strategy
   - SEO and accessibility checklist
   - Project timeline tracker

3. **html14_Rubric.md** (165 lines)
   - Comprehensive 100-point grading rubric
   - 7 categories covering all course competencies:
     - HTML Structure & Semantic Markup (20 pts)
     - CSS Styling & Design (20 pts)
     - Responsive Design (15 pts)
     - JavaScript & Forms (15 pts)
     - Content & Media (10 pts)
     - SEO & Accessibility (10 pts)
     - Deployment & Testing (10 pts)
   - Detailed point breakdowns and descriptors
   - Common deductions and extra credit opportunities

4. **html14_SelfAssessment.md** (204 lines)
   - Student self-assessment checklist
   - Mirror of project requirements
   - Reflection questions for student learning
   - Self-rating scale with justification
   - Feedback section for instructor notes

### Example Solution (Reference Website)

A complete, professional example website for a fictional coffee shop: **Mountain View Coffee Co.**

#### HTML Pages (4 pages + proper structure)

5. **html14_ExampleSolution_index.html** (183 lines)
   - Home page with hero section, features, and testimonials
   - Demonstrates proper semantic HTML5 structure
   - Navigation menu and footer with links
   - Multiple content sections with proper hierarchy
   - Fully commented for student learning

6. **html14_ExampleSolution_about.html** (231 lines)
   - About page with company story and mission
   - Team member profiles in grid layout
   - Core values section with emoji icons
   - Timeline of company history
   - Multiple content types (text, images, lists)

7. **html14_ExampleSolution_menu.html** (311 lines)
   - Menu page with multiple product categories
   - **REQUIRED TABLE ELEMENT**: Coffee comparison guide
   - 6 columns x 6 rows with headers and data
   - Product cards with descriptions and prices
   - Demonstrates proper table structure and styling
   - Menu sections with article elements

8. **html14_ExampleSolution_contact.html** (214 lines)
   - Contact page with contact form
   - **REQUIRED FORM ELEMENT**: Contact form with multiple inputs
   - 6 form fields with validation attributes
   - Contact information cards
   - Map placeholder image
   - Additional connection options

#### CSS Stylesheet

9. **html14_ExampleSolution_styles.css** (1,099 lines)
   - Professional, well-organized stylesheet
   - CSS Custom Properties (variables) for colors, fonts, spacing
   - Mobile-first responsive design approach
   - Media queries for tablet (768px) and mobile (480px)
   - Comprehensive component styling:
     - Header with sticky positioning
     - Hamburger menu for mobile navigation
     - Grid layouts for multiple components
     - Form styling with error states
     - Table styling with alternating rows
     - Footer with responsive grid
   - Smooth transitions and hover effects
   - Accessibility-first approach with color contrast
   - Print styles for accessibility

#### JavaScript File

10. **html14_ExampleSolution_script.js** (347 lines)
    - Form validation with comprehensive error handling
    - **REQUIRED FORM VALIDATION**: 
      - Name validation (2+ characters)
      - Email format validation (regex pattern)
      - Phone number validation (10+ digits, optional)
      - Subject/dropdown validation
      - Message validation (10+ characters minimum)
    - Error messaging and DOM updates
    - Real-time validation on blur events
    - Success message display
    - Form reset on successful submission
    - Hamburger menu toggle functionality
    - Mobile navigation close on link click
    - Smooth scrolling for anchor links
    - Accessibility enhancements (Escape key to close menu)
    - Fully documented with comments

## How to Use These Files

### For Instructors

1. **Use CapstoneSpec.md** to understand project scope and timeline
2. **Share PlanningTemplate.md** with students in Session 1
3. **Use Rubric.md** for grading student work
4. **Reference the example solution** to show students what excellence looks like
5. **Distribute SelfAssessment.md** at the end of the project

### For Students

1. **Read CapstoneSpec.md** to understand all requirements
2. **Complete PlanningTemplate.md** before starting to code
3. **Reference the example solution files** when stuck:
   - Look at the HTML structure for page organization
   - Study the CSS for styling techniques and responsive design
   - Review the JavaScript for form validation patterns
4. **Use the rubric** to self-evaluate your work
5. **Complete SelfAssessment.md** before submitting

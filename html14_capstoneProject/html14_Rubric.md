## 3. Responsive Design (15 points)

### Understanding Strand 6.3: Create mobile-first responsive layouts

| Criteria | Excellent (15) | Good (13) | Satisfactory (11) | Needs Work (8) | Missing (0) |
|----------|----------------|-----------|-------------------|-----------------|-------------|
| **Mobile Design** | Website looks excellent on mobile (<600px); single-column layout; readable text; properly sized buttons; optimized navigation | Website works on mobile; mostly single-column; readable; minor layout issues | Website functional on mobile; readable but some design issues | Website barely works on mobile; text too small or layout broken | Not mobile-responsive |
| **Tablet Design** | Seamless adaptation from mobile to tablet (600-1024px); good use of two-column layout where appropriate | Works well on tablet; appropriate layout changes; minor visual issues | Functional on tablet; some awkward spacing or layout | Limited tablet optimization; some elements misaligned | Poor tablet experience |
| **Desktop Design** | Excellent layout on desktop (1024px+); multi-column where appropriate; professional use of space | Works well on desktop; good layout and spacing; minor issues | Adequate desktop layout; some wasted space or poor alignment | Desktop layout functional but unattractive | Poor desktop design |
| **Media Queries or Framework** | Proper CSS media queries for all breakpoints (mobile-first approach) OR professional use of Bootstrap Grid; responsive images using srcset or <picture> | Media queries implemented for major breakpoints; responsive images used | Media queries present; covers 1-2 breakpoints; basic responsive images | Limited media queries; static image sizes | No responsive design |

## Common Deductions

- **Missing semantic HTML tags:** -2 to -5 points
- **Broken links or navigation:** -2 to -5 points
- **Form validation doesn't work:** -3 to -5 points
- **Not responsive on mobile:** -5 to -10 points
- **Missing alt text on images:** -1 to -3 points
- **HTML or CSS validation errors:** -1 per error (up to -5)
- **Not deployed to GitHub Pages:** -5 to -10 points
- **Missing README or planning template:** -2 to -5 points

## Extra Credit Opportunities

- **+5 points:** Accessibility score of 95+ on Lighthouse
- **+5 points:** Advanced interactivity (multi-page image gallery, smooth scroll animations, dark mode toggle, etc.)
- **+3 points:** Performance optimization (images optimized, lazy loading, minified CSS/JS)
- **+3 points:** Advanced CSS (CSS Grid, Flexbox mastery, CSS animations beyond basic)
- **+2 points:** Custom domain or professional deployment beyond GitHub Pages
- **+2 points:** SEO optimization (structured data/schema.org, sitemap.xml, robots.txt)

Table with colspan (Column Spanning)

Merge cells across columns using the colspan attribute.

colspan Example: <td colspan="2"> merges 2 columns
___

Table with rowspan (Row Spanning)

Merge cells across rows using the rowspan attribute.

rowspan Example: <td rowspan="2"> merges 2 rows

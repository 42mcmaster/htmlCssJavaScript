Image Best Practices

  • Alt text: Describe the image for screen readers and accessibility
  • File format: Use JPG for photos, PNG for graphics with transparency, SVG for icons
  • File size: Compress images to improve page loading speed
  • Width/Height: Define dimensions to prevent layout shift when image loads
  • Responsive: Use CSS to make images scale on mobile devices

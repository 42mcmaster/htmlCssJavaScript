# DIY Task 13: Full Project Deployment & Testing

**Duration:** 90 minutes (can span 2-3 class periods)
**Objective:** Take any previous project, fully validate and test it, and deploy it live to GitHub Pages.

---

## Overview

This is your capstone task for Lesson 13. You will:

1. **Select** one of your best projects from Units 1-4
2. **Validate** the HTML and CSS code
3. **Fix** all errors found
4. **Test** in multiple browsers and on mobile
5. **Deploy** to GitHub Pages
6. **Document** your testing and deployment process

At the end, you'll have a live, professional-quality website on the internet.

### Step 4: Enable GitHub Pages

1. Go to your repository Settings
2. Find "Pages" in the sidebar
3. Select "main" branch, "/root" folder
4. Click Save
5. Wait 1-2 minutes

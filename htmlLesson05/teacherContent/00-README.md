## 🎯 Lesson Overview

**Core Concept:** 
Forms collect data → Data has types → Types determine analysis → Good design = clean data

**Your Strength:** 
80% of the lesson is data analysis (your comfort zone), 20% is HTML preview (brief)

**Student Activity:** 
Students help design a Google Form about lunch preferences, fill it out, then analyze the data together

**Why This Works:**
- Hands-on and engaging
- Plays to your Excel expertise
- Introduces forms conceptually (not deep coding)
- Shows real-world application
- Bridges to database curriculum

---

## ⏱️ Lesson Timeline

| Time | Section | What You Do |
|------|---------|-------------|
| 0-5 min | Hook | Show examples, introduce pipeline |
| 5-15 min | Data Types | Teach text, number, category, date, boolean |
| 15-30 min | Build Form | Create Google Form as a class |
| 30-35 min | HTML Preview | Show HTML form, connect concepts |
| 35-45 min | Analysis | Analyze form data in Excel/Sheets |
| 45-50 min | Wrap Up | Reflection, key takeaways |

## 💡 Teaching Tips

### Do This:
✅ **Ask questions** - Make it interactive ("What data type should this be?")
✅ **Think aloud** - Narrate your choices ("I'm using a dropdown BECAUSE...")
✅ **Connect to real world** - Reference DoorDash, Netflix, surveys
✅ **Use your Excel skills** - This is your strength, lean into it
✅ **Stay on time** - Follow the timeline, use quick reference card

### Avoid This:
❌ **Don't dive deep into HTML** - It's just a preview, keep it brief
❌ **Don't wing it** - Follow the script, especially for timing
❌ **Don't skip the analysis** - That's your payoff moment
❌ **Don't let form building drag** - 15 minutes max
❌ **Don't apologize** - You know this content, be confident

## 🆘 Emergency Backup Plans

### If Google Form won't load:
→ Use pre-made sample dataset
→ Paper form planning activity
→ Focus more on Excel analysis

### If students can't access form:
→ Fill it out as a class from teacher computer
→ Use fake data you input yourself
→ Continue with pre-made dataset

### If technology completely fails:
→ Do paper-based form design activity
→ Discuss data types with examples
→ Show printed screenshots of analysis

### If running way over time:
→ Skip detailed form building (show pre-made)
→ Do ONE analysis task instead of multiple
→ Shorten HTML section to 2 minutes

================================================================================
TEACHING TIPS
================================================================================

1. START WITH CONSOLE: Spend time in Developer Tools (F12) before HTML
2. EMPHASIZE DEBUGGING: console.log is their best friend for testing
3. MODERN FIRST: Teach let/const first, mention var for recognition
4. TEMPLATE LITERALS: Students love the readability; highlight this
5. FUNCTION REUSE: Show how one function saves repetition
6. MAD LIBS MOTIVATION: Keep story generator project fun and creative
7. PAIR PROGRAMMING: Consider partners for Tasks 09a and 09b
8. DEMO FIRST: Always show working solution before students attempt
9. ERRORS ARE OK: Uncomment error examples to show TypeError messages
10. EXTEND STRETCH: Advanced students should tackle stretch goals
